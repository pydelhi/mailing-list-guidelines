Mailing List Guidelines

Here are some pointers which you need to keep in mind

  • Always use a search engine before you ask a question.
  • DO NOT use CAPITAL LETTERS as it is like shouting.
  • Do NOT post questions with No Subject or very long Subject line.
  • Do NOT send HTML mails to mailing list. Also don't send Dual-format mails containing both text and html.
  • DO NOT use SMS Texts like u for YOU etc.
  • DO NOT hijack discussion thread, Always start a fresh thread for discussion something new.
  • Trim properly while replying , Follow interleaved or Bottom style of posting to the list.
  • [Commercial] tag for mails related to Jobs.
  • [OT]use common sense, while using this Tag for any info which is Off Topic
  • Avoid cross posting.
  • Avoid sending attachments to the mailing list.
  • Use http://gist.github.com/ for posting Output or Error log.
  • Download Shakthi Kannan's Presentation here.
  • How to Ask Questions http://www.catb.org/~esr/faqs/smart-questions.html
